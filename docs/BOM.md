## Notes

- The component kit (#17) covers all resistors and capacitors needed: 10kΩ drain resistor, 1kΩ source resistor, 100kΩ feedback resistor, 100nF decoupling caps, 100µF LED cap, 10µF output DC-blocking cap, 4.7kΩ LED resistor, 300Ω WS2812B data resistor.
- Use only one of the 5 KY-040 encoders and one of the 5 SPST buttons per build; spares are handy.
- Use only one of the 3 OLED modules per build.
- The shielded cable is used for the guitar input run (jack → JFET gate) to minimize hum.
